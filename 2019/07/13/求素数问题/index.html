求素数问题

判断101-200之间有多少个素数，并输出所有素数。

知识拓展：sqrt函数

头文件：math.h

函数原型：double sqrt(double x)

返回值：X的平方值

这里 i 是 int 类型要强制转换成 double型，但如果只写成 sqrt(i+1)也是可以的，int型可以自动转换成double型，这里使用 i+1而不是 i是为了让 sqrt的计算结果更大，不至于遗漏。
