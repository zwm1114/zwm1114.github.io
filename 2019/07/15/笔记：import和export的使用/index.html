一、import

1.导入整个模块的内容

import * as myModule from '/modules/my-module.js';

如果上面导入的模块包含一个接口doAllTheAmazingThings()，可以这样调用：

myModule.doAllTheAmazingThings();

2.导入单个接口

import {myExport} from '/modules/my-module.js';

myExport已经从模块my-module导出（因为整个模块被导出）或显式地导出（使用export语句），将myExport插入当前作用域。

3.导入多个接口

import {foo, bar} from '/modules/my-module.js';

同样，这些接口都已经从my-module中导出了。

4.导入带有别名的接口

即对接口重命名

导入多个带有别名的接口，与导入多个接口同样，不再赘述。

5.作为副作用导入一个模块

import '/modules/my-module.js';

这将运行模块中的全局代码, 但实际上不导入任何值。

6.导入默认值

当引入模块默认导出对象、函数、类等，用import语句导入默认接口。

1.直接导入默认值

import myDefault from '/modules/my-module.js';

二、export

在导出多个值时，命名导出非常有用。在导入期间，必须使用相应对象的相同名称。但是，可以使用任何名称导入默认导出，例如：

从导入模块导出默认值

export {default} from 'mod';

不可以使用：
